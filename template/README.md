##

### Setup environment

```
node >= v12.x
yarn >= 1.2x
```

### start development

#### install dependencies

```
yarn install
```

#### starting development

```
yarn start
```

#### production build

```
yarn build
```

### Technologies

- React v17
- React Router V5
- ReduxJS - Redux-toolkit
- Antd
- Styled-components
- Create-react-app
